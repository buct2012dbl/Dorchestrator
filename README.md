# Agent Orchestrator

MacOS Agent Orchestrator for Claude Coding Agent

## Features

- Graph View: Visual workflow management with React Flow
- Terminal View: Real-time agent execution logs with xterm.js
- Agent Orchestration: Task scheduling and lifecycle management

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Architecture

- Electron for desktop app framework
- React for UI components
- React Flow for graph visualization
- xterm.js for terminal emulation
